# 재고 예측 대시보드 배포 패키지

## 📦 배포 파일 위치

`dist` 폴더의 모든 파일을 함께 복사하여 공유하세요.

## 🚀 사용 방법

### 방법 1: 간단한 웹 서버 실행 (권장)

1. `dist` 폴더로 이동
2. `start-server.bat` 파일을 더블클릭
3. 브라우저에서 `http://localhost:8000` 으로 접속

### 방법 2: 직접 열기

- `dist/index.html` 파일을 브라우저로 직접 열기
- ⚠️ 일부 브라우저에서 CORS 정책으로 인해 제한될 수 있습니다.

### 방법 3: 웹 서버에 업로드

1. `dist` 폴더의 모든 파일을 웹 서버에 업로드
2. `index.html`로 접속

## 📁 필수 파일

- `index.html` - 메인 HTML 파일
- `assets/` 폴더 - JavaScript 및 CSS 파일
- `miyawng-bini-data.json` - 제품 데이터

## ⚙️ AI 분석 기능 사용

AI 분석 기능을 사용하려면:
1. 웹 서버에서 실행 중인 경우, 서버 환경 변수에 `VITE_GEMINI_API_KEY` 설정
2. 또는 브라우저 콘솔에서 직접 API 키를 입력하는 방식으로 수정 필요

## 📝 참고사항

- 모든 파일을 함께 복사해야 정상 작동합니다.
- 파일 경로는 상대 경로로 설정되어 있습니다.
- 로컬에서 직접 열 때는 일부 기능이 제한될 수 있습니다.
